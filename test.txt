Step 1 - Install create-react-app
C:\Users\Tutorialspoint\Projects>npx create-react-app my-app


Step 2 - Delete all the source files
> cd my-app/src
my-app/src > del *
my-app/src\*, Are you sure (Y/N) ? y


Step 3 - Add files (index.js and index.css)
my-app/src type nul > index.css
my-app/src type nul > index.js


Step 4 - Add following code to index.js
import React from 'react';
import ReactDOM from 'react-dom';
import './index.css';


Step 5 - Run project
npm start
